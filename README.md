# TechPortal Dashboard

A modern and responsive dashboard website developed using **HTML, CSS, and JavaScript**. The project provides a professional user interface with a navigation bar, sidebar menu, content sections, and popup authentication forms for Login and Sign Up.

## Features

* Responsive Navigation Bar
* Sidebar Dashboard Menu
* Interactive Hero Section
* Information Cards
* Login Popup Form
* Sign Up Popup Form
* Clean and Modern User Interface
* Easy-to-Understand Code Structure
* Beginner-Friendly Project

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your preferred web browser.

### Using VS Code Live Server

1. Open the project in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click on `index.html`.
4. Select **Open with Live Server**.
